## 功能说明

本Skill提供完整的数据可视化能力，支持以下功能：

### 核心功能

1. **智能图表推荐**
   - 根据数据特征自动推荐最适合的图表类型
   - 时间序列数据 → 折线图
   - 分类对比数据 → 柱状图
   - 占比分析数据 → 饼图
   - 关联分析数据 → 散点图

2. **多图表类型支持**
   - 折线图：趋势展示、时间序列
   - 柱状图：分类对比、数量统计
   - 饼图：占比分析、比例展示
   - 散点图：关联分析、分布展示
   - 雷达图：多维对比、能力评估
   - 热力图：密度展示、矩阵数据

3. **数据自动处理**
   - JSON数据解析
   - CSV数据转换
   - 数据格式校验
   - 自动计算聚合

4. **专业样式输出**
   - 默认商务配色方案
   - 响应式图表尺寸
   - 清晰的图例和标签
   - 支持导出PNG图片


## 执行流程

### Step 1: 数据解析
解析用户提供的原始数据，识别数据结构和类型：
- 如果数据以文本形式提供，解析JSON/CSV格式
- 如果数据较复杂，调用数据分析脚本进行预处理

### Step 2: 图表类型推荐
根据数据特征推荐最佳图表类型：
- 时间序列（日期/月份）→ 折线图
- 分类字段 + 数值字段 → 柱状图
- 单一数值字段（百分比/比例）→ 饼图
- 双数值字段 → 散点图

### Step 3: ECharts配置生成
生成符合ECharts规范的配置对象：
```javascript
{
  title: { text: '图表标题' },
  tooltip: {},
  xAxis: { type: 'category', data: [...] },
  yAxis: { type: 'value' },
  series: [{ type: 'bar', data: [...] }]
}
```

### Step 4: HTML渲染输出
生成自包含的HTML文件，包含：
- ECharts CDN引入
- 图表配置
- 响应式容器
- 自动渲染脚本

### Step 5: 图片导出（如需要）
可选调用图片导出功能，生成PNG格式图表

## 注意事项

1. 数据格式：优先使用JSON数组格式 `[{key: value}, ...]`
2. 数据量：建议单图表数据点不超过1000个
3. 复杂需求：多系列图表或特殊图表类型可能需要更多配置
